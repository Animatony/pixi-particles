Class: Emitter

Emitter

Emitter

new PIXI.particles.Emitter(particleParent, particleImages, config)

Emitter.js, line 24

A particle emitter.

Name Type Description
particleParent PIXI.Container

The container to add the
particles to.
particleImages Array | PIXI.Texture | String optional

A texture or array of textures to use
for the particles. Strings will be turned
into textures via Texture.fromImage().
config Object optional

A configuration object containing settings for the emitter.

Name Type Default Description
emit Boolean true optional

If config.emit is explicitly passed as false, the Emitter
will start disabled.
autoUpdate Boolean false optional

If config.emit is explicitly passed as true, the Emitter
will automatically call update via the PIXI shared ticker.
